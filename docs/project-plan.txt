개인프로젝트 - 중고차 통합 플랫폼

1. 프로젝트 목적
중고차 거래, 차량관리, 주유소/정비소 검색, 커뮤니티, 알림, 추천, 이탈위험 관리를 포함한 개인 포트폴리오용 웹 플랫폼을 만든다.

2. 기술스택
Frontend: React, Vite, React Router, Axios
Backend: Spring Boot, Java 21, Spring Security, JPA
Database: PostgreSQL
Model: Python, FastAPI, pandas, scikit-learn
Map: Kakao Map API

3. 주요 기능
- 회원 기능
- 중고차 매물 CRUD
- 찜하기
- 판매자 문의
- 1:1 메시지
- 알림
- 매물 추천
- 주유소/정비소 검색
- 차량관리툴
- 커뮤니티
- 관리자 페이지
- 이탈위험 계산

4. 개발 순서
1단계: 프로젝트 구조 정리
2단계: 프론트 실행 확인
3단계: 프론트 라우터/페이지 구조 정리
4단계: 백엔드 기본 프로젝트 설정
5단계: DB 설계
6단계: 회원 기능
7단계: 중고차 매물 기능
8단계: 메시지/알림 기능
9단계: 차량관리툴
10단계: 관리자 기능
11단계: 추천/이탈위험 모델 연동
