# 오토케어 개인프로젝트 프론트엔드

중고차 팀프로젝트 압축파일을 그대로 이어가지 않고, 개인프로젝트 방향에 맞게 필요한 구조만 새로 정리한 React/Vite 프로젝트입니다.

## 포함 기능

- 중고차거래: 일반 중고거래 목록, 상세, 등록 화면
- 주유소/정비소: 카카오맵 API 연결 전 화면 구조, 정비소 상세 및 리뷰 화면
- 차량관리툴: 보유 차량 주행거리와 마지막 점검일 기준 알림
- 커뮤니티: 일반게시판, 차량관리팁, 질문게시판
- 로그인/회원가입: localStorage 기반 임시 로그인

## 제거한 방향

- 경매 시스템
- 회사/딜러 관리 구조
- 관리자 이탈관리
- 최종거래/정산

## 실행 방법

```bash
npm install
npm run dev
```
